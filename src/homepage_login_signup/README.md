# Homepage, Login & Signup Module

This module contains all components and pages related to user authentication and the homepage of the Code Clash Arena application.

## Directory Structure

- **components/**: Reusable UI components used across login, signup, and homepage pages
- **pages/**: Main page components for homepage, login, and signup screens
- **style/**: CSS and styling files specific to this module
- **utilities/**: Helper functions and utility code for authentication and form handling

## Features

- User login functionality
- User registration/signup
- Homepage landing page
- Authentication flow management

## Usage

Import components and pages from this module to integrate authentication features into the main application.

## Dependencies

- React
- React DOM
