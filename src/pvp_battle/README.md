# PvP Battle Module

This module contains components and pages for real-time player-versus-player competitive coding battles.

## Directory Structure

- **components/**: Reusable UI components for PvP battle features
- **pages/**: Main page components for battle lobbies, battle arenas, and results
- **style/**: CSS and styling files specific to this module
- **utilities/**: Helper functions and utility code for battle mechanics and real-time updates

## Features

- Battle matchmaking and lobbies
- Real-time coding competition interface
- Live code execution and comparison
- Real-time score tracking and updates
- Battle timer and round management
- Post-battle statistics and rankings

## Usage

Import components and pages from this module to integrate PvP battle features into the main application.

## Dependencies

- React
- React DOM
- WebSocket (for real-time communication)
