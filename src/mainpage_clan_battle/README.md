# Main Page & Clan Battle Module

This module contains components and pages related to the main application interface and clan-based battle features.

## Directory Structure

- **components/**: Reusable UI components for the main page and clan battle features
- **pages/**: Main page components for dashboard, clan battles, and related screens
- **style/**: CSS and styling files specific to this module
- **utilities/**: Helper functions and utility code for clan management and battle logic

## Features

- Main dashboard/home page
- Clan management interface
- Clan battle mechanics
- Team coordination features
- Battle matchmaking and tracking

## Usage

Import components and pages from this module to integrate clan battle features into the main application.

## Dependencies

- React
- React DOM
