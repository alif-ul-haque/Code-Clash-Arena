# Components - Main Page & Clan Battle

Reusable UI components for the main page and clan battle module.

## Purpose

This directory contains modular, reusable React components that are used across the main page and clan battle features.

## Component Categories

- **Clan Components**: Clan cards, member lists, clan stats
- **Battle Components**: Battle cards, participant lists, score displays
- **Dashboard Components**: Charts, widgets, summary cards
- **UI Components**: Tables, modals, notification badges

## Guidelines

- Keep components small and focused on a single responsibility
- Use props for configuration and data passing
- Maintain consistent styling with the module's style guide
- Document component props and usage

## Examples

Each component should export a single default React component and can include prop types or TypeScript definitions.
