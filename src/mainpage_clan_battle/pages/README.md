# Pages - Main Page & Clan Battle

Main page components for the main page and clan battle module.

## Purpose

This directory contains the full page components that represent distinct routes/screens in the main application:

## Pages

- **Main Dashboard**: Central hub displaying user stats and quick actions
- **Clan List**: Browse and manage clans
- **Clan Details**: View clan information and members
- **Clan Battle**: Clan vs Clan battle interface
- **Battle Results**: Display battle outcomes and statistics

## Page Structure

Each page component should:
- Import necessary sub-components from the `components/` directory
- Import styling from the `style/` directory
- Use utilities from the `utilities/` directory for business logic
- Handle page-level state and routing

## Navigation

Pages are typically connected through React Router or similar routing mechanism in the main application.
