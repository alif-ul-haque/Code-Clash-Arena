# Styles - Practice Gym

Styling files for the practice gym module.

## Purpose

This directory contains CSS files and styling utilities specific to the practice gym module.

## File Organization

- **CSS Modules**: Component-specific styles
- **Global Styles**: Shared styles used across the module
- **Variables**: Color schemes, typography, spacing constants
- **Editor Styles**: Code editor theme and syntax highlighting

## Styling Approach

- Use modular CSS for component-specific styles
- Define reusable utility classes for common patterns
- Maintain consistency with the application's design system
- Use semantic class naming conventions

## Tips

- Separate concerns: keep component styles separate from layout styles
- Consider performance by minimizing CSS file size
- Use CSS variables for theming and dynamic styling
- Include dark/light theme support for code editor
